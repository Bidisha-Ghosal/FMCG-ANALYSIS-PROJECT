# 📊 FMCG Consumer Behavior & Inflation Analysis

## 📌 Project Overview

This project analyzes how inflation and agricultural trends influence consumer purchasing behavior in the FMCG (Fast-Moving Consumer Goods) sector. The goal is to identify patterns in revenue growth, consumer preferences, and the impact of economic factors such as CPI and rural–urban population distribution.

---

## 🎯 Objectives

* Understand the relationship between inflation and FMCG sales
* Analyze how agriculture impacts consumer demand
* Identify which brands and product categories perform better under economic pressure
* Compare rural vs urban consumption patterns

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, Matplotlib) – Data cleaning & analysis
* **SQL** – Querying structured datasets
* **Power BI** – Dashboard visualization
* **Excel** – Initial data handling

---

## 📂 Dataset Description

The project uses multiple datasets, including:

* FMCG company performance data
* Inflation (CPI) data
* Agriculture-related indicators
* Population distribution (Rural vs Urban)

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values and standardizing formats
* Revenue growth vs inflation comparison
* CPI trend analysis
* Rural vs Urban consumption behavior
* Brand and product-level insights

---

## 📈 Key Insights

* Inflation has a direct impact on FMCG purchasing patterns
* Rural consumption is more sensitive to agricultural performance
* Certain FMCG categories remain stable despite inflation
* Urban consumers show relatively consistent demand trends

---

## 📊 Dashboard

Power BI dashboard includes:

* Revenue trends
* Inflation vs consumption comparison
* Regional consumption patterns
* Brand performance analysis

---

## 🚀 How to Run the Project

1. Open the Jupyter Notebook (`.ipynb`)
2. Run all cells sequentially
3. Load datasets from the provided files
4. Use SQL queries for deeper analysis
5. View dashboard in Power BI

---

## 📌 Conclusion

This project demonstrates how macroeconomic factors like inflation and agriculture significantly influence consumer behavior in the FMCG sector. The insights can help businesses make better pricing, distribution, and marketing decisions.
